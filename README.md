# 웹 게임 서비스를 위한 AWS Data Management Architecture

## 프로젝트 소개
데이터는 무결성이 지켜져야 한다.
실시간으로 빠르게 데이터가 변하는 게임에서는
더욱 신경써야 하는 부분이다.
회원 정보와 게임 데이터는 외부 데이터베이스에서 관리하고
필요한 게임 데이터만 RDS에 저장하여 사용하며
게임로그를 S3에 저장해
외부 데이터베이스를 업데이트하는 시스템을 구현

## 구현 목표
게임 데이터의 무결성을 지키기 위한 웹 게임 데이터 관리 아키텍처 구축 (RDS, S3, Lambda)
OpneVPN, Site-to-SiteVPN을 활용한 안전한 원격 접속 환경 구축
ALB, AutoScaling을 이용한 고가용성 탄력적 운영
ACM과 WAF를 활용한 웹서비스 보안
동물 키우기(다마고치) 웹 게임 서비스 구현
